## ğŸ”§ Customization

Want to make it your own? These are easy edits inside `index.html`.

- Edit messages  
  Find and modify the JavaScript message arrays:
```javascript
const messages = [
  "Your custom message here",
  // Add more messages...
];
```

- Change colors  
  Modify the CSS gradient in the `<style>` section:
```css
background: linear-gradient(135deg, #your-color-1, #your-color-2);
```

- Replace audio  
  Swap the audio file URL in the audio tag:
```html
<source src="your-music-url.mp3" type="audio/mpeg">
```
